# Spotify Clone

A simple **Spotify page clone** built using **HTML** and **CSS** as part of my early web development learning journey.

A Spotify-inspired UI clone built using HTML and CSS for learning front-end development.

## About the Project

This project was created to practice the fundamentals of front-end web development, including:

* Semantic HTML structure
* CSS styling and layouts
* Flexbox positioning
* Responsive design basics
* UI recreation from an existing website

The goal of this project was to improve my understanding of HTML and CSS by recreating the look and feel of Spotify's interface.

## Technologies Used

* HTML5
* CSS3

## Features

* Clean Spotify-inspired user interface
* Responsive layout
* Modern styling with CSS
* Beginner-friendly project structure

## Learning Outcomes

While building this project, I learned how to:

* Structure web pages using semantic HTML
* Style components with CSS
* Use Flexbox for layouts
* Organize project files
* Recreate a real-world website design
